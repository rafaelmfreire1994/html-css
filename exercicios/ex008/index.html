Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag < B > (não semântica).

Nesta frase, tenho um termo em destaque usando a tag < STRONG > (com semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag < I > (não semântica).

Nesta frase, temos um termo em ênfase usando a tag < EM >

Texto marcado

Podemos criar também um texto marcado usando a tag < MARK >.

Também podemos mudar a cor da marcação utilizando CSS com a tag STYLE, como essa.

Textos pequenos e grande

Estamos criando um texto grande e um pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele dever ser lido, mas não considerado utilizando a tag < del >.

Texto inserido

Podemos marcar um texto inserido para dar uma ênfase e indicar que ele foi adicionado depois utilizando a tag < ins >.

Existe também o sublinhado com a tag < U > (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3 selecionamos a cédula e colocamos dentro da tag < sup >.

Texto subscrito

Para inserir coisas do tipo H2O selecionamos a cédula e colocamos dentro da tag < sub >.
